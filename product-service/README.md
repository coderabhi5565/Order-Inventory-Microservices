# Product Service  

Ye microservice product data ko manage karta hai.

## Features
- Create product  
- Update product  
- Soft delete product  
- Fetch by ID / SKU  
- List with pagination & filters  
- Bulk fetch for Order-Service

## Tech Stack
- Java  
- Spring Boot  
- REST API  
- SQL Database  

## Folder Structure
- controller → endpoints  
- service → business logic  
- repository → database access  
- entity → database model  
- dto → request/response models  
- exception → custom errors
